Week 3: Application Types

Learn about different types of applications! What do they have in common?

Concepts:

  • What do different application types have in common?
    • User Interface: The way a user interacts with the application or website
    • Functionality: Whether a design works and helps the user meet their goals and needs.
    • Data Storage: Applications store data for many reasons, such as studying and analyzing trends.
    • Security: Prevents secret vulnerabilities against threats like unauthorized access.
    • Performance: How the app is functioning.

Application:

These are examples commonly used applications:

  • Desktop Applications
  • Web Applications
  • Mobile Applications
  • Cloud Applications
  • Enterprise Applications
  • Gaming Applications

Quiz:

► Question 1: What are 2 things applications have in common?

User Interface, Functionality, Data Storage, Security, and/or Performance
